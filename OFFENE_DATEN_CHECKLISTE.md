# Daten-MVP: Bitte einmal schicken 🙌

## Rechtliches:

1. Name der Fahrschule (genau so, wie er offiziell geführt wird)
2. Inhaber:in / vertretungsberechtigte Person
3. Anschrift (Straße, Hausnummer, PLZ, Ort)
4. Telefonnummer
5. E-Mail-Adresse
6. Zuständige Aufsichtsbehörde (Name + Ort/Adresse)
7. Handelsregister + Registernummer (falls vorhanden)
8. USt-IdNr. / Wirtschafts-Identifikationsnummer (falls vorhanden)


## Für den Website Content:

- Standorte (jeweils mit Adresse)
- Öffnungszeiten (gesamt oder je Standort)
- Namen (Voll / Spitzname, wie es euch lieber ist) + Rollenbeschreibung (beispiel: Inhaber / oder Klasse B+A etc) + kurze Texte für Team/Über-uns
